説明：wacomドライバを再起動するバッチファイル(Windows専用)です．

使用方法
　1.wacom_rebootフォルダを任意の場所に配置します．
  2.フォルダ内”reboot.bat”もしくは”reboot[Minimized]”を実行します．
　3.数秒後ドライバの再起動が完了します．
　
　reboot.batを右クリックから”ショートカットの作成”を行うことでコンピュータのどの場所にも配置することが可能です．

　　※デフォルトの構成では自動的に管理者権限で起動するようになっています．
　　※reboot[Minimized]はウィンドウを最小化した状態で実行します．


動作について
　・https://www.pixiv.net/novel/show.php?id=6262620を参考に作成いたしました．ありがとうございます！一部挙動の変更と追加を行っています。
　・cmd.lnkを介することで管理者権限での実行を実現しています．
　・各wacomサービスの再起動に加えて，Wacomデスクトップセンターを終了するようにしています．

注意事項
　極稀に再起動に失敗することがありますので，その際はもう一度実行してください．
　筆圧を使用しているソフトウェアを実行している場合は一度終了してください．
